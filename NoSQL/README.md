# NoSQL

## Description

Projet consacré aux bases de données **NoSQL**, permettant de stocker et manipuler des données non relationnelles de manière flexible et scalable.

## Objectifs

* Comprendre les différences entre SQL et NoSQL
* Manipuler des bases de données orientées documents
* Effectuer des requêtes de base (CRUD)
* Gérer des données non structurées

## Concepts clés

* Bases de données non relationnelles
* Documents (JSON)
* Collections
* CRUD (Create, Read, Update, Delete)

## Technologies

* MongoDB
* Python 3 (optionnel)

## Exécution

```bash
mongo < fichier.js
```

## Bonnes pratiques

* Structurer les données selon les besoins applicatifs
* Éviter les duplications excessives
* Optimiser les requêtes avec des index
