# Pagination

## Description

Projet consacré à l’implémentation de la **pagination** pour gérer et afficher efficacement de grandes quantités de données.

## Objectifs

* Comprendre le principe de pagination
* Implémenter une pagination simple (page / taille)
* Gérer les offsets et limites
* Optimiser les performances côté API

## Concepts clés

* Offset / Limit
* Page-based pagination
* Cursor-based pagination (introduction)
* Performance et scalabilité

## Technologies

* Python 3
* API / Backend

## Exécution

```bash
python3 nom_du_fichier.py
```

## Bonnes pratiques

* Limiter la taille des pages
* Éviter les requêtes coûteuses
* Prévoir une pagination cohérente et stable
